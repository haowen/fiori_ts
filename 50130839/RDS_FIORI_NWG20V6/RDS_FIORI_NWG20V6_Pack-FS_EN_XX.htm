The SAP Fiori apps rapid-deployment solution features prepackaged, instant-value apps across multiple business functions. This package offers a choice to deploy one or many SAP Fiori apps with a predictable price and time with wide variety of optional services to choose from.

It allows now to deploy ANY SAP Fiori app, which was delivered by SAP, including SAP Fiori apps requiring the SAP HANA platform as a prerequisite.

The package will provide extra information and guidance on:

  • Extensibility: step by step documentation of how to hide, rename and add a field to a SAP Fiori app
  • Prerequisite checks for the customizing of an existing SAP backend system for some selected Fiori apps
  • Used SAP backend transactions specific for each SAP Fiori app
  • Using the SAP Fiori Reference Library
  • Integrate SAP Fiori apps in the SAP Enterprise Portal

This rapid-deployment solution can be implemented together with SAP Fiori Design rapid-deployment solution.
