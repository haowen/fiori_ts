Backend Prerequisite Checks

Purpose

The purpose of the Prerequisite Checks is to provide a set of questions to help the prospect to decide if the solution would work in the custom installation. Detailed checks help answering the questions.

Walk through the questions on the Prerequisite Checks tab and give an answer in the corresponding drop-down box. If you need further details on the questions, see the respective details on the other tabs. You can stop your evaluation if the result field shows a finding.

Objective

  • Ensure that the system meets technical requirements
  • Ensure that the system meets all the application prerequisites (master data, application configuration)

Target Audience

  • Sales / Presales / Application Consultant in collaboration with the customer to get a clear view on the prerequisites which need to be fulfilled in the customer installation
