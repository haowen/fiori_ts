Web Application Icons

The following is a list of currently available icons for the SAP Enterprise Portal 6.0. The icons are offered

  • as single gif files, which you can download using the browser and re-use
  • and as a complete ZIP archive

They are grouped into the following categories:

  • File type
  • Folder type
  • Object type
  • Generic actions
  • Helper apps
  • Collaboration
  • Text / HTML editing (toolbar icons)
  • Sequence
  • Affecting status
  • Status icons with definable meanings
  • Status icons with specific meanings

The standard size for portal application icons is 14*14 pixels.
Exception: Toolbar icons are 12*12 pixels.
