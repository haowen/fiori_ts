SAP Fiori apps rapid-deployment solution V6.20

Companies running SAP can now deploy SAP Fiori apps that provide great user experience to support business tasks on many devices. Using best practices, service providers install the infrastructure. Quickly and predictably, companies can boost employee productivity and reduce total cost of ownership.

    Use the SAP Fiori Advanced Security Setup in order to incease your security level around Fiori. Find a description of how to implement Single Sign On (SSO) with SAML 2.0 or SSO with X.509.

  • SAP Fiori Integration with SAP Enterprise Portal

    This optional scope item will allow to integrate some specific SAP Fiori app into the SAP Enterprise Portal. You can either integrate the complete Fiori Launchpad or only a single SAP Fiori app which can be accessed in a tab section.

    • More information

    Fiori apps run as a web-based application in your preferred Internet Browser. If you are looking into embedding the Web-App into a native application like the Fiori Client, then use this guide on how to do this step by step.
